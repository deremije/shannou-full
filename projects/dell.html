DELL OS THEME & APPS

THE PROBLEM

Dell produced a few models of smart phones which they wanted to differentiate in the market by offering features to complement and seamlessly integrate with those which were already bundled on their desktop computers.

MY ROLE

I was in charge of visual and production design to create apps and desktop widgets for Dell's new line of Android phones. I additionally created a theme so that the look and feel of the embedded OS would have consistency with the existing visual style.

Themed OS Visual Design Samples

I created a design for almost every part of the Android operating system in order to give it a unique theme which would be consistent with the rest of the design language. I created visual designs and custom launcher icons for the different desktop widgets which were embedded on the devices. To ensure that my image assets and specs were implemented according to design, I worked closely with the development teams.

Dell Mobile Apps

To apply the design language which I created for the mobile OS, I also designed the user experience for custom embedded apps. These included a calendar app, a social media photo app, an app for viewing and recording TV and one which both played movies on the device or functioned as as a remote control for desktops.
